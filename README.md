# StickerVault 🎨

A full-stack sticker platform where users can upload stickers and admins approve them.

## 🚀 Features

* Upload stickers
* Admin approval system
* Categories & subcategories
* Download system

## 🛠 Tech Stack

* React + TypeScript
* Node.js + Express

## 📌 Future Improvements

* Firebase integration
* User authentication
* Trending stickers system
